Project Rebranding: flowrs â†’ floxide¶

Overview¶

This document summarizes the rebranding process from "flowrs" to "floxide" that was completed on February 26, 2025.

Rebranding Steps¶

  1. Name Selection
  2. Selected "floxide" as the new name, combining "flow" (workflow) and "oxide" (a reference to Rust)

  3. Verified availability on crates.io

  4. Codebase Updates

  5. Updated all package names in Cargo.toml files
  6. Renamed crate directories from flowrs-* to floxide-*
  7. Updated all code references, including imports and error types

  8. Updated documentation references

  9. Repository Changes

  10. Renamed repository directory from flow-rs to floxide

  11. Updated Git remote URL to point to the new repository name

  12. Documentation

  13. Created ADR-0032 to document the rebranding decision
  14. Updated README.md and other documentation files

  15. Updated API documentation file names

  16. Cleanup

  17. Removed old directories and files
  18. Updated references in the site directory

Verification¶

The rebranding was verified through: - Successful build with cargo build - Successful test run with cargo test - Grep searches to ensure no remaining references to the old name

Next Steps¶

  • Update GitHub repository name
  • Update crates.io package names when publishing
  • Notify users of the name change
  • Update external documentation and links

References¶

  • ADR-0032: Project Rebranding to Floxide
